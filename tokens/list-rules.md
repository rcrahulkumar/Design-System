List System Rules:

Types:
- Text → simple data
- Icon → action-based lists
- Image → visual context
- Status → metadata-heavy (time, badge)
- Radio → single selection
- Checkbox → multi selection
- Action → inline actions

Guidelines:
- Keep items scannable
- Use consistent spacing
- Avoid too many elements in one item
- Use badges for status
- Use CTA only when needed

Structure:
- Title required
- Description optional
- Meta optional (time, value)
- CTA optional

Usage:
- Dashboard → status list
- Forms → radio/checkbox list
- Navigation → icon list
- AI → recommendation list
