Checkbox Rules:

Behavior:
- Multiple selections allowed
- Independent toggles

States:
- Default → unchecked
- Selected → checked
- Disabled → not interactive

Structure:
- Label required
- Description optional

Guidelines:
- Use for multi-select only
- Use radio for single select
- Keep labels short
- Group related options

Usage:
- Symptoms → multi-select
- Preferences → multiple choices
- Forms → multiple answers
- AI flows → flexible selection
