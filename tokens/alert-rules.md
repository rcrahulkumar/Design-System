Alert System Rules:

Types:
- Small → inline feedback
- Large → page-level communication

States:
- Info → guidance
- Success → confirmation
- Warning → caution
- Error → failure

Guidelines:
- Always include icon for clarity
- Use CTA only when action is required
- Keep message concise
- Avoid stacking multiple alerts

Usage:
- Forms → error/success alerts
- Dashboard → small alerts
- Onboarding → info alerts
- AI → contextual alerts
