Badge System Rules:

Types:
- NoBG → inline text
- Light → default usage
- Outline → minimal UI
- Dark → high emphasis
- Icon → quick visual signal

States:
- Success → completed, verified
- Warning → attention needed
- Error → failure, blocked
- Inactive → neutral, scheduled

Guidelines:
- Use badges for status, not actions
- Keep text short (1–2 words)
- Prefer light variant for most UI
- Use dark only in high emphasis areas

Usage:
- Dashboard → light badges
- Tables → outline badges
- AI insights → NoBG or light
- Alerts → dark badges
