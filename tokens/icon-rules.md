Icon System Rules:

Principles:
- Use icons to support, not replace text
- Maintain consistent size per context
- Keep visual weight balanced

Guidelines:
- Buttons → icon + text
- Inputs → small icons
- Lists → medium icons
- Banners → large icons

Do not:
- Mix multiple icon styles
- Use too many icons in one view
- Replace labels with icons alone

Categories:
- Health → domain-specific icons
- Actions → system icons
- Status → semantic icons
