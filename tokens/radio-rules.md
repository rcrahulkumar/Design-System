Radio Button Rules:

Behavior:
- Only one option selectable
- Always part of a group

States:
- Default → not selected
- Selected → active choice
- Disabled → not interactive

Structure:
- Label required
- Description optional

Guidelines:
- Use for single-choice decisions
- Use checkboxes for multi-select
- Keep options clear and short
- Avoid more than 5–6 options

Usage:
- Forms → gender, duration
- Health input → symptom severity
- AI flows → guided selection
