Chip System Rules:

Types:
- Selection → interactive (filters, toggles)
- Tag → informational (non-clickable)

States:
- Default → available
- Selected → active choice
- Disabled → unavailable

Guidelines:
- Use chips for quick decisions
- Allow multi-select for filters
- Keep labels short (1–2 words)
- Avoid overcrowding UI with too many chips

Usage:
- Filters → selection chips
- AI insights → tag chips
- Dashboard labels → tag chips
